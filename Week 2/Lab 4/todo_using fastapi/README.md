# To-Do List REST API

A simple, in-memory RESTful API built with **FastAPI** to manage a To-Do list. This project allows users to create, read, update, and delete tasks.

##  Technologies Used
* **Python 3**
* **FastAPI** (Web framework)
* **Uvicorn** (ASGI server)
* **Pydantic** (Data validation)
