# To-Do List REST API

A simple RESTful API for managing todo tasks built with Flask.

## Features

- Create new tasks
- View all tasks
- View specific task by ID
- Update existing tasks (title and completion status)
- Delete tasks
- Swagger UI documentation
- No database required - stores data in memory
